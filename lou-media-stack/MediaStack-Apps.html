üîß Quick Setup Guide

Priority Setup Order:

  1. WireGuard: Configure VPN server for secure remote access
  2. PostgreSQL & Valkey: Ensure databases are healthy and backed up
  3. Portainer: Set up admin account and container monitoring
  4. Authentik: Configure Single Sign-On for security
  5. Traefik: Set up reverse proxy and SSL certificates
  6. Gluetun: Configure VPN client for download protection
  7. Jackett: Add indexers for content discovery
  8. Deluge: Configure torrent client (change default password)
  9. Radarr/Sonarr/Lidarr: Connect to indexers and download clients
  10. Jellyfin/Plex: Set up media libraries and transcoding
  11. Overseerr/Jellyseerr: Configure request management
  12. Recyclarr: Sync quality profiles using TRaSH guides
  13. Dashboards: Customize Heimdall or Homepage for easy access
