üéØ Quick Start Recommendation

For your 99-year-old grandmother's ease of use, configure services in this order:

  1. Traefik - Ensure SSL and routing works
  2. Plex - Set up media libraries
  3. Jackett - Add a few reliable indexers
  4. Deluge - Basic download client setup
  5. Sonarr & Radarr - Connect everything for automation
  6. Overseerr - Simple request interface
  7. Heimdall Dashboard - One-click access to everything

Once these core services are working, grandmother can easily request shows/movies through Overseerr, and everything will be automatically downloaded and added to Plex!
