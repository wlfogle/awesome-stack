# Media Stack Services and Recommended OS

## Core Infrastructure:
1. **Postgres** - Linux (Alpine recommended for lightweight containers)
2. **Valkey** - Linux (Alpine)
3. **Gluetun** - Linux (Alpine)
4. **Traefik** - Linux (Alpine)
5. **Authentik Server & Worker** - Linux (Debian/Ubuntu)

## Essential Media Services:
6. **Prowlarr** - Linux (Ubuntu/Debian)
7. **Jackett** - Linux (Ubuntu/Debian)
8. **Flaresolverr** - Linux (Debian)
9. **qBittorrent & Deluge** - Linux (Ubuntu for GUI, Alpine for server)
10. **Sonarr, Radarr, Lidarr, Readarr, Mylar3, Whisparr** - Linux (Ubuntu/Debian)

## Media Servers:
11. **Jellyfin, Plex, Emby** - Linux (Ubuntu for wide support)
12. **Audiobookshelf, Calibre-Web** - Linux (Debian)
13. **IPTV-Proxy, Tvheadend** - Linux (Ubuntu/Debian)

## Enhancement Services:
14. **Bazarr, Overseerr, Jellyseerr, Ombi** - Linux (Debian)
15. **Tautulli, Kometa, Gaps** - Linux (Debian/Ubuntu)

## Monitoring & Analytics:
16. **Prometheus, Grafana** - Linux (Alpine)
17. **Exportarr, Checkrr** - Linux (Debian/Ubuntu)

## Management & Utilities:
18. **Portainer, Filebot** - Linux (Alpine for Docker management)
19. **Flexget, Buildarr** - Linux (Debian)
20. **Vaultwarden** - Linux (Debian)

## Security & Access:
21. **Crowdsec, Tailscale** - Linux (Debian)

## Dashboards & Maintenance:
22. **Organizr, Homarr, Homepage** - Linux (Ubuntu)
23. **Watchtower, Recyclarr, Unpackerr** - Linux (Alpine)
